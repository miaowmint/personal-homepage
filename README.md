## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/miaowmint/personal-homepage.git
```

2. 打开 `index.html` 即可在浏览器中查看

## 自定义配置

你可以通过修改以下变量来个性化你的主页：

### 基本信息
- 昵称：修改 HTML 中的 `.nickname` 元素
- 简介：修改 HTML 中的 `.bio` 元素
- 头像：修改 HTML 中 `.avatar img` 的 `src` 属性
- 背景：修改 CSS 中 `.background` 的 `background-image` 属性

### 社交链接
在 HTML 的 `.social-links` 部分修改链接地址
